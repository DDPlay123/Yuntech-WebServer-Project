# Yuntech-Web Server-Project
雲科畢業專題「Web Server」部分

開發深度學習模型，同時開發手機App及網頁伺服器。手機app主要功能為辨識雲科的生物，並將辨識出的資料及圖片傳送到網站。網站接收手機的資料及圖片後，可用於地圖顯示當下拍攝時間地點及種類。

Android連結：<https://github.com/DDPlay123/Yuntech-Android-Project>
